# Adrian Cosma

I am an AI researcher working on language models, multilinguality, tokenization, semantic generalization, and interpretable systems.

My research is concerned with how learning systems acquire structure: how representations emerge, how linguistic structure transfers across languages, and how we can build systems that expose useful abstractions rather than merely optimize opaque predictors.

## Research interests

- Language models and multilingual generalization
- Tokenization and subword structure
- Synthetic languages and controlled semantic evaluation
- Automatic feature discovery and interpretability
- Tool-using and self-improving AI systems

## Selected work

- **In-vitro cross-lingual transfer in language models**  
  Controlled experiments on how language similarity, corpus composition, and tokenization affect multilingual transfer.

- **Automatic prompt optimization for dataset-level feature discovery**  
  A framework for discovering interpretable textual features that improve downstream supervised objectives.

- **Vision-language model robustness under perceptual color distortions**  
  Programmatic visual illusions for testing brittleness in VLM perception.
